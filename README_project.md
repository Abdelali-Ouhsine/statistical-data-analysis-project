# 📊 Statistical Analysis Project – Health & Automotive Data

## 🧾 Project Overview
As a Junior Data Analyst, this project focuses on applying statistical concepts to real-world business problems.

### 🎯 Mission
- Understand key statistical concepts
- Apply them to real datasets
- Produce reliable and actionable insights

### 📌 Use Cases
- 🏥 Public Health Data Analysis
- 🚗 Automotive Data Analysis (MPG Dataset)

---

## 🎯 Learning Objectives
By the end of this project, I was able to:

- Define and explain key statistical concepts
- Distinguish between discrete and continuous variables
- Perform descriptive statistics
- Create advanced visualizations
- Detect and interpret outliers
- Measure relationships between variables (correlation & hypothesis testing)
- Apply statistical tests and justify conclusions
- Connect statistical theory to real business use cases

## 🗂️ Project Structure

### 🧠 Phase 1 – Understanding & Documentation
Created clear and structured documentation for:

- Descriptive statistics (mean, median, variance, standard deviation)
- Data visualization (histogram, boxplot)
- Population vs Sample
- Probability distributions:
  - Bernoulli
  - Binomial
  - Poisson
  - Normal
  - Student
  - Chi-square
- Central Limit Theorem (CLT)
- Sampling & estimation
- Hypothesis testing (H0, H1, p-value)
